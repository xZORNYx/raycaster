# Raycaster software renderer

- ⚡ Old school software renderer
- 🔫 Raycasting rendering
- ✝️ Written purely in C just as god intended


## Build 🏗️
```shell
# SDL2 library required
$ git clone https: https://github.com/xZORNYx/raycaster.git
$ cd raycaster
$ make
$ ./raycaster
```
